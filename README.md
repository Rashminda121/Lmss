# Lumina LMSS – Learning Management Support System Enhanced with AI Assistance 🎓💡

Lumina LMSS is a full-featured learning platform built to streamline course management, foster engagement, and support both students and lecturers through a centralized AI-powered system.

## 🚀 Key Features

- ✅ Course Enrollment & Management  
- ✅ AI-Powered Assistance for Students & Lecturers  
- ✅ Custom Dashboards with Engagement & Performance Insights  
- ✅ Interactive and Centralized Learning Environment  
- ✅ Role-Based Access Control  
- ✅ Email Notifications  
- ✅ Scalability & Adaptability  

## 🧠 AI Features

- AI-based support for students and lecturers powered by **DeepSeek**
- Context-aware learning suggestions
- Smart insights and summarization tools

## 🛠️ Tech Stack

### Frontend
- **Next.js**  
- **Tailwind CSS**  
- **TypeScript**

### Backend
- **Node.js**, **Express.js**, **REST API**
- **Axios** for API communication

### Database
- **MongoDB**
- **MySQL**
- **Prisma ORM**

### AI Integration
- **DeepSeek**

### CI/CD & Version Control
- **GitHub Actions**, **GitHub**

### Containerization & Deployment
- **Docker**
- **AWS EC2 Instances**

### Authentication & File Uploads
- **Clerk**
- **UploadThing**

### Testing & Utilities
- **Jest**
- **Nodemailer**
- **JWT (JSON Web Token)**

## 📬 Email Notifications

- Automatic email alerts for enrollment, assignments, and updates via **Nodemailer**

## 🔐 Role-Based Access

- Lecturers, Students, and Admins have different permissions and interfaces
